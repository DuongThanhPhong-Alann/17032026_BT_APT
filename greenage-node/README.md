# GreenAge header (Node.js + Express + Pug)

Only builds the header area like the screenshot (no `.html` files).

## Run

```bash
cd greenage-node
npm install
npm run dev
```

Open `http://localhost:3000`.

## Slideshow

Put images into `greenage-node/public/images/slideshow/` (e.g. `01.jpg`, `02.jpg`, ...). The homepage auto-rotates them.
